== Description ==

If a user scrolls to the bottom of a post, redirect him to the homepage.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp_scrolltobottomredirect` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Scroll to the bottom of a post to try out.
4. edit style.css in your plugins folder to change styles
